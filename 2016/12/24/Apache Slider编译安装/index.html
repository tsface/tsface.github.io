Apache Slider 编译安装

一、系统要求

  1. 系统编译要求

    • Maven
    • Linux

  2. 系统安装要求

    • Apache Hadoop Hadoop 2.6+(2.6.0会编译失败)
    • Apache Zookeeper 3.4+
    • Oracle JDK 1.7 (64-bit)
    • Python 2.6+
    • openssl

二、编译

  1. 下载 Apache Slider，最新版本是0.91.0
  2. 解压到指定文件下编译
  3. 获取编译生成的安装包
  4. 获取安装包(二选一)

三、安装

  1. 在本地解压slider-0.91.0-incubating-all.tar.gz安装包，建立软连接
  2. 添加本地环境变量
  3. 修改Slider相关脚本的环境变量slider/conf/slider-env.sh

四、测试

  1. slider version
